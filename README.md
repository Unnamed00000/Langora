# Langora

Langora is a browser-based language learning app for starting from zero and moving toward confident conversation.

## Languages

- Japanese
- Russian
- Georgian
- German
- Danish
- Italian
- English

## Course Structure

The app is organized as a complete learning path:

- A0: alphabet, writing system, pronunciation, and first words
- A1: daily words and present tense
- A2: present, past, and future together
- A3: questions, answers, and simple dialogues
- B1: conversational level
- B2: opinions, details, and stronger grammar
- C1: natural speech and fluency

## Run

Open `index.html` in a browser, or serve the folder with any static file server.
